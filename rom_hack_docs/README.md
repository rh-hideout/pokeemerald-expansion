# Pokemon Brazilianite - ROM Hack Design Documents

This folder contains all design documentation for the Pokemon Brazilianite ROM hack, built on pokeemerald-expansion.

## Document Index

| File | Purpose |
|------|---------|
| [overview.md](overview.md) | High-level vision, themes, and goals |
| [story.md](story.md) | Plot, characters, and narrative beats |
| [region_and_maps.md](region_and_maps.md) | Region design, new/modified maps, routes |
| [pokemon_changes.md](pokemon_changes.md) | New species, stat changes, type changes, movesets |
| [moves_and_abilities.md](moves_and_abilities.md) | New/modified moves and abilities |
| [trainers.md](trainers.md) | Gym leaders, rivals, E4, boss fights, trainer teams |
| [wild_encounters.md](wild_encounters.md) | Wild encounter tables by route/area |
| [items.md](items.md) | New/modified items and item locations |
| [events_and_scripts.md](events_and_scripts.md) | Scripted events, side quests, key moments |
| [difficulty_and_progression.md](difficulty_and_progression.md) | Level curve, caps, scaling, QoL |
| [config_changes.md](config_changes.md) | Expansion config flags to toggle |
| [aesthetic_changes.md](aesthetic_changes.md) | Tilesets, sprites, music, UI changes |
| [changelog.md](changelog.md) | Running log of changes made |

## How to Use

Fill in each document as you plan your hack. Each file has a template with sections and examples to guide you. You don't need to complete everything at once - flesh out documents as ideas develop.
