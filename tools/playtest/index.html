Debug Menu Quick Reference

  • Overworld Debug: Hold R (S key) + press Start (Enter key)
  • Battle Debug: Press Select (Backspace key) during battle
  • Sprite Viewer: Press Select (Backspace key) in summary screen
  • Warp anywhere: Overworld Debug → Utilities → Fly to map / Warp to map
  • Get any Pokemon: Overworld Debug → Give → Pokemon
  • Heal party: Overworld Debug → Party → Heal party
  • Set flags/badges: Overworld Debug → Flags
