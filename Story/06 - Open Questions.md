# Open Questions

## Purpose

Keep unresolved decisions visible instead of resolving them through prose, code, maps, art, or this documentation set.

## Current Canon

The following are unresolved:

- How Zorua learns the history of the Great Illusion.
- Zorua's relationship to Zoroark, if any.
- Whether and when Zorua evolves; it is not tied causally to the epilogue.
- Which individual trainers return and each reunion outcome.
- Which professor, town, and tutorial details are retained, rewritten, or removed.
- **Open Question — Lab context:** who used the lab, and how is history presented there?
- How Zorua's initial ability to use a human illusion is presented without chosen-one framing.
- Which story-critical habitats gate the main finale.

## Notes

The handoff and canonical Spec are authoritative when a listed question is resolved by an approved decision.
